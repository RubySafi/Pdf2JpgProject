1. Abstract

PDF2JPG is a utility that allows you to easily convert PDF files into JPG images. The app offers three different ways to perform the conversion: via a command-line interface (CLI), a batch file, or a graphical user interface (GUI).

You can use the app by selecting a PDF file and converting it to JPG images. The resulting images will be placed in the specified output directory or the default directory if none is provided.

2. Setup

The release folder is self-contained and ready for use. To launch the application, please follow these steps:

  • Set up a Python environment if you don't have one.
  • Install the necessary Python libraries pdf2image and tqdm using pip install pdf2image tqdm.
  • Install the .NET Framework if it's not already installed.
  • Manually add the poppler folder inside the release directory. (This is required for PDF rendering.)

Once these steps are completed, the app should be ready to use.

Uninstall

To uninstall, simply delete the release folder. No additional uninstallation steps are needed unless you manually installed Python or the .NET Framework.

Usage Summary

There are three ways to convert a PDF file to JPG:

  • CLI Approach: Use the command-line interface to run the Python script.
  • BAT File Approach: Use the batch file to quickly convert PDFs.
  • GUI Approach: Use the graphical user interface to select files and convert them.

In both the BAT file and GUI approaches, the resulting JPG files will be automatically opened in the file explorer.

Default Output Directory

The default output directory is an output folder that will automatically be created inside the release folder. You can change the output directory if needed.

Usage Details

Usage Detail 1: CLI Approach

The command to run the Python script is as follows:

python pdf2jpg.py <pdf-file-path> [output-directory-path]

Note: The output-directory-path is optional. If omitted, the default output directory will be used.

Usage Detail 2: BAT File Approach

To convert a PDF file using the batch file:

  1. Drag and drop the PDF file onto the run_pdf2jpg.bat file.
  2. The PDF will be automatically converted to JPG images, with the output saved in the default output directory.

Usage Detail 3: GUI Approach

To use the GUI:

  1. Launch Pdf2JpgGui.exe.
  2. A form will appear where you can select the PDF file and the output directory (optional).
  3. Click the "Convert" button to start the conversion.
  4. After the conversion, the resulting JPG files will automatically open in the output directory.

You can also drag and drop a PDF file onto the form to initiate the conversion.
